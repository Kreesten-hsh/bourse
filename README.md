# to the world

Private mobility command center for collecting, scoring, reading, tracking, and applying to funded international opportunities.

## Stack

- Frontend: Next.js, TypeScript, Tailwind CSS, Phosphor Icons, Motion
- Backend: FastAPI, Pydantic, PostgreSQL-ready structure

## Current Milestone

Phase 1 foundation:

- Next.js app shell with premium visual direction.
- FastAPI health endpoint.
- Product and design documents.
- MVP specification and implementation plan.
